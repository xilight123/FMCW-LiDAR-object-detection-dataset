# FMCW-object-detection-dataset
FMCW LiDAR Point Cloud of object detection dataset based on CARLA

## Dataset structure
    FMCWLidDet_Dataset
    |
    |-- bags
    |    |-- Town 01.bag
    |    |-- Town 02.bag
    |    |-- ... 
    |-- pcds
    |    |-- 000001.pcd
    |    |-- 000002.pcd
    |    |-- ...
    |-- bins
    |    |-- 000001.bin
    |    |-- 000002.bin
    |    |-- ...
    |-- labels
    |    |-- 000001.txt
    |    |-- 000002.txt
    |    |-- ...
    |-- ImageSets
       |-- train.txt
       |-- val.txt
       |--test.txt
